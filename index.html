Mode d'emploi

Gère ta boutique en quelques clics, sans toucher au code

🔑

Accéder à l'admin

Clique sur le petit ⚙ discret en haut à droite. Entre ton mot de passe admin. Une barre noire apparaît en haut.

➕

Ajouter un produit

En mode admin, clique sur + Ajouter un produit. Remplis nom, description, prix, catégorie. Clique Enregistrer.

✏️

Modifier un produit

En mode admin, passe la souris sur une carte. Deux boutons apparaissent. Clique Modifier pour éditer.

🗑️

Supprimer un produit

En mode admin, survole une carte et clique Supprimer. Une confirmation est demandée avant la suppression.

💾

Sauvegarder sur GitHub

Les produits sont sauvegardés dans ton navigateur. Pour les rendre permanents, re-uploade le fichier sur GitHub après chaque modification.

🔒

Changer le mot de passe

Dans le code, cherche admin123 et change-le. Re-uploade sur GitHub. Fais-le dès que possible !
